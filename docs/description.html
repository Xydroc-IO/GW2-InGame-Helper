GW2 In-Game Helper

In-game sites. One DLL. No memory reads.

Version 1.2.5 · Signature 0x48454C50 · Author xydroc · License MIT · Requires Raidcore Nexus + Guild Wars 2

GW2 In-Game Helper is a Raidcore Nexus addon that opens useful Guild Wars 2 websites inside the game. Browse the Wiki, Snowcrows, MetaBattle, gw2efficiency, Fast Farming, and the official site without alt-tabbing — all from one modular overlay browser.

Players install a single DLL. The Chromium helper is embedded and extracts on first use. The browser runtime comes from the game’s own bin64/cef. No WebView2, no winetricks, no separate CEF download.

Designed to stay policy-safe: official Nexus APIs only. No Guild Wars 2 process memory reads, no game hooks, and no injected click/chat macros for item lookup.

Replaces the older GW2-InGameWiki and GW2-InGame-Snowcrow addons. Disable those if they are still installed.

Install

  1. Close Guild Wars 2.
  2. Copy GW2-InGame-Helper.dll into your game’s addons folder.
  3. Start the game and enable GW2-InGame-Helper in Nexus (Ctrl+O) if needed.
  4. Restart if Nexus asks you to.

That is the entire install. Do not ship or download any extra .exe or CEF package.

How to use

  • Ctrl+Shift+H (or K) — open / close the helper, or click the QuickAccess icon.
  • Use the site dropdown to switch between Wiki, builds, tools, and farming sites.
  • Toolbar: Home, Back, Forward, Reload.
  • Click outside the window on the game world to return movement and skills to Guild Wars 2.

Keybinds are rebindable in Nexus under KB_HELPER_TOGGLE and KB_HELPER_ITEM.

Wiki item lookup

  1. In game, Shift+Click an item so a [&…] chat link appears (or copy one from chat).
  2. Copy it with Ctrl+C.
  3. Press Ctrl+Shift+U.
  4. The helper opens the Wiki for that item.

Lookup is clipboard-only. The addon does not fake Shift+Click, open chat, or press Enter (those macros caused mount / party-chat side effects).

Features

  • Categorized in-game site picker
  • Nexus QuickAccess top-bar icon
  • Built-in how-to homepage on first open
  • Wiki item lookup from clipboard [&…] chat links
  • Saved window size, position, opacity, and font scale
  • Works on Windows and on Linux through Wine/Proton
  • Single-file player install (DLL only)

What this addon does not do

  • Does not read or write Guild Wars 2 process memory
  • Does not hook or patch game code
  • Does not automate combat, inventory, trading, or economy
  • Does not bot, macro unattended play, or spoof packets
  • Does not inject mouse/keyboard macros to capture items

Requirements

  • Guild Wars 2 (64-bit Windows client)
  • Raidcore Nexus
  • Game CEF present at bin64/cef/libcef.dll (shipped with the game)
