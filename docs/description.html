GW2 In-Game Helper

In-game sites & Discords. One DLL. No memory reads.

Version 1.7.7.0 · Signature 0x48454C50 · Author xydroc · License MIT · Requires Raidcore Nexus + Guild Wars 2 · Updates GitHub Releases

GW2 In-Game Helper opens useful Guild Wars 2 websites and community Discord invites inside the game — Wiki, builds, tools, guides, farming, and more.

Players install a single DLL. No memory reads; official Nexus APIs only.

Install: copy GW2-InGame-Helper.dll to <GW2>/addons/ only. Runtime files extract into <GW2>/addons/GW2-InGame-Helper/. Open with Ctrl+Shift+H or the QuickAccess icon.
