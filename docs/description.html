GW2 In-Game Helper

In-game sites. One DLL. No memory reads.

Version 1.3.0 · Signature 0x48454C50 · Author xydroc · License MIT · Requires Raidcore Nexus + Guild Wars 2

GW2 In-Game Helper is a Raidcore Nexus addon that opens useful Guild Wars 2 websites inside the game. Browse the Wiki, Snowcrows, MetaBattle, gw2efficiency, Fast Farming, and the official site without alt-tabbing — all from one modular overlay browser.

Players install a single DLL. The Chromium helper is embedded and extracts on first use. The browser runtime comes from the game’s own bin64/cef.

Official Nexus APIs only. No Guild Wars 2 process memory reads, no game hooks, and no item-lookup hotkeys or input macros.

Replaces the older GW2-InGameWiki and GW2-InGame-Snowcrow addons. Disable those if they are still installed. Item-lookup binds from older Helper builds are removed on load.

Install

  1. Close Guild Wars 2.
  2. Copy GW2-InGame-Helper.dll into your game’s addons folder.
  3. Start the game and enable GW2-InGame-Helper in Nexus (Ctrl+O) if needed.
  4. If an old KB_HELPER_ITEM bind remains in Nexus, delete it.

How to use

  • Ctrl+Shift+H (or K) — open / close, or click the QuickAccess icon.
  • Use the site dropdown to switch between Wiki, builds, tools, and farming sites.
  • Toolbar: Home, Back, Forward, Reload.
  • Click outside the window on the game world to return movement and skills.

Rebind the toggle in Nexus under KB_HELPER_TOGGLE.
